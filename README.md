# LiveBleWeb

LiveBleWeb is a lightweight web-based interface designed for BLE (Bluetooth Low Energy) device communication and image transfer workflows. The project is focused on real-time interaction with BLE-enabled hardware devices and provides a simple frontend for testing and managing Bluetooth-connected systems.

## Features

* BLE device interaction
* Simple and responsive UI
* Image transfer support
* Firebase Hosting deployment ready
* Lightweight HTML/CSS/JavaScript architecture
* Fast loading and minimal dependencies

# Firebase Hosting Deployment

## Install Firebase CLI

```bash
npm install -g firebase-tools
```

## Login

```bash
firebase login
```

## Initialize Hosting

```bash
firebase init hosting --project epaper-30f1b-web
```

Use:

```text
Public Directory: .
Single Page App: No
```

## Deploy

```bash
firebase deploy
```

# Local Development

You can run the project locally using any simple web server.

Example:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```
